# Kombi-BeyazEsya-Website#

❄️ Çeşme Klima Teknik Servis VE Kombi-BeyazEsya-Website - Landing Page

Bu proje, İzmir/Çeşme bölgesinde hizmet veren bir klima teknik servisi için optimize edilmiş, yüksek performanslı ve SEO odaklı bir "Landing Page" çözümüdür.
Site Linki : https://www.cesmeklimaservisi.tr/

## 🚀 Öne Çıkan Özellikler
- **Hız:** Next.js ve Vercel Edge Network sayesinde ışık hızında yükleme süreleri.
- **Dönüşüm Odaklı:** Müşterilerin tek tıkla WhatsApp veya Telefon üzerinden ulaşabileceği optimize edilmiş butonlar.
- **SEO Dostu:** Search Console doğrulaması yapılmış, %100 Lighthouse SEO skoruna sahip yapı.
- **Vibe Coding:** AI destekli geliştirme süreçleri (Cursor, Claude, Gemini) ile 24 saatten kısa sürede canlıya alma (deployment).

## 🛠️ Teknik Stack
- **Framework:** [Next.js 14+](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

- **Best Practices:** 100/100
- **SEO:** 100/100
